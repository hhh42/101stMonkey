Virtual Networking

My basic problem is that I’m setting up a small private cloud using MAAS with mostly virtual machines attached. I’d like to have multiple machines hosting multiple virtual networks of KVM machines. All mananged with virt-manager. I’m currently good at one machine I just have to figure out how to tunnel too or bridge .. or route.. or.. whatever network magic is missing.

I intend to write about my solution here.

Quick update.. I hate networking. A near solution I figured out is to have separate virtual networks running a few KVM based VMS on separate boxes. Instead of running MAAS inside the internal network I run it with two network cards. One real one wired into it’s own switch that some day will have some real hardware on it and a virtual network where the VMs live. I then put a static route to each of the virtual networks (and physical) on the physical router that is the gateway for the MAAS controller and the rest of the host machines in the house.
